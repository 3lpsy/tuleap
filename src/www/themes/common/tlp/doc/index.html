Layout

Framed content .tlp-framed

By default there isn't margins on the main container; this allows you to be able to use the full with of the viewport (for example when you use tabs).
However most of the time you will need to add some margins (for example when you use panes). This is achieving by using .tlp-framed.

Use .tlp-framed-vertically or .tlp-framed-horizontally if you need to add margin only at the top/bottom or to the left/right of the container.

Use all of them to produce complex layout. Show / hide margins

Centered content .tlp-centered

In order to center content, you can use the .tlp-centered on the container.

You can also combine .tlp-centered with .tlp-framed to add margins.

Tables .tlp-table

Good practices:

  • Use .tlp-table-actions on a bloc element before the table if needed (e.g. "+ Add" button…).
  • Use .tlp-table-cell-numeric on numeric cells to right align numeric value (header must be aligned with the content, so apply this class also on the th).
  • Use .tlp-table-cell-actions on action cells to right align the buttons and to prevent the wrapping. No title is required for this kind of column.

Forms .tlp-form-element

Good practices:

  • When it is suitable, use the good input type (password, number, email, search, url, …).
  • To disable a form element, use .tlp-form-element-disabled and disabled attribute.
  • To set a form element required, use required attribute and add <i class="fa fa-asterisk"></i> in the label.
  • To put a form element in error, use .tlp-form-element-error.
  • By default, form elements are displayed using width: 100%. You can override this by setting a custom width in css or by adding:
    • the size attribute on inputs when it makes sens (e.g. size="4" for a year)
    • the cols attribute on textareas when it makes sens (you can also play with rows attribute)
    • the .tlp-select-adjusted class on selects and multi-selects

Modals .tlp-modal

Structure

Usage

  • When there are both cancel and primary action buttons, then the primary button is always on the right.
  • To be able to use large buttons on the footer you should use the class tlp-modal-footer-large.
  • Buttons that have data-dismiss="modal" attribute in the modal will close it.
  • Click on the backdrop or on the escape key will also close the modal.
  • You can remove closing modal on the escape key pressed: var the_modal = tlp.modal('element-id', {keyboard: false} );
  • You must explicitly call the modal in javascript: var the_modal = tlp.modal('element-id');. This will return a Modal instance. You can then call show(), hide(), or toggle() methods on this instance.
  • On the the_modal.show() method an event tlp-modal-shown is dispatched.
  • On the the_modal.hide() method an event tlp-modal-hidden is dispatched.
