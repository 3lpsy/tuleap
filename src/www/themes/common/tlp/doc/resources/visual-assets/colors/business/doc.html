To iterate on those colors, use the following SCSS snippet:

@each $color-name, $colors in $tlp-swatch-color-map {
    // This will create classes like .inca-silver, .deep-blue, etc.
    &.#{$color-name} {
        border-bottom-color: map-get($colors, 'primary');
        // You can also use the 'secondary' color
    }
}
