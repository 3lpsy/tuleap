2.2 Connect your CodeX server

To connect your CodeX Server, open the CodeX perspective and

  1. select the "Add a new server" icon in the CodeX view ,

  2. then fill the server configuration form:

    1. Name: the name that will appear in your Eclipse interface

    2. URL: the URL of your CodeX server (e.g: http://codex.xerox.com;)

    3. Username: your CodeX username

    4. Password: your CodeX password (Warning: if you give your password, it will be stored in your computer. If you don't want to, leave it blank, your password will be asked you each time you want to connect the CodeX server)

  3. apply the changes,

  4. then click the OK button.

Your CodeX server is now stored in your Eclipse preferences. You can browse your projects and trackers (See Section 2.3).
